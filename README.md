# OmniYield 🌐🤖

**OmniYield** is an advanced AI-powered DeFi Delta-Neutral Yield Advisor. By utilizing the fast and powerful Gemma 3 model, it allows users to simply chat with an intelligent assistant to explore, simulate, and execute cross-chain yield farming strategies seamlessly and safely.

## The Future of DeFi Interactions
OmniYield bridges the complexity gap in Decentralized Finance by providing a conversational interface for portfolio management. Whether you're interacting with real BSC testnet smart contracts via MetaMask or aggregating mock multi-chain data (Arbitrum, Solana), OmniYield abstracts away the complicated DeFi analytics into an intuitive, chat-driven experience.

### Key Features
- **Conversational Yield Discovery**: Simply ask the AI to find you the best yield across different networks.
- **Predictive AI Allocation**: Dynamic AI probability scoring and real-time risk assessment for Delta-Neutral strategies.
- **Omni-Chain Portfolio Management**: Unified dashboard showing your aggregated assets across various chains.
- **One-Click Execution**: Validate AI transaction breakdowns and execute trades directly through the Chat UI via the unified Wallet Control panel.
